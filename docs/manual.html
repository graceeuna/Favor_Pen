🖍️ ScreenPen Portable 사용 설명서

화면 위 어디에든 그리고 주석하는 무설치 항상-위 오버레이 도구 · Windows 10/11

v0.3 · 단일 실행파일 · 설치 불필요
소개 실행·다른PC 툴바 그리기 도구 도형·텍스트·번호 모드·효과 타이머 마우스 하이라이트 단축키 팁·주의

1. 제품 소개

ScreenPen Portable은 바탕화면·발표 슬라이드·영상·웹·앱 어떤 화면 위에든 즉시 펜·도형·텍스트로 주석을 그리는 도구입니다. 필요하면 화면을 통과(click-through)시켜 아래 앱을 그대로 조작할 수 있고, 설치 없이 실행파일 하나로 동작합니다.

  • 펜 · 하이라이터 · 지우개, 색상 · 굵기 자유 설정
  • 도형(직선·화살표·사각형·타원) · 텍스트 · 번호 태그
  • 화이트보드/블랙보드 · 고스트 모드 · 페이딩 잉크 · 마우스 하이라이트 · 돋보기 · 타이머
  • 실행취소/다시, 스크린샷(PNG+클립보드), 트레이 상주, 설정 자동 저장
  • 듀얼·다중 모니터 지원

2. 실행 & 다른 PC에서 사용하기

이 앱은 무설치(Portable)입니다. 실행파일을 더블클릭만 하면 됩니다. 설치·관리자 권한·레지스트리 변경이 필요 없습니다.

두 가지 배포 형태

형태크기특징추천
자체 포함
(self-contained)		 약 71 MB .NET 런타임까지 내장 → 아무 PC·USB에서 바로 실행 처음 보는 PC, USB 휴대
런타임 의존
(framework-dependent)		 약 1.6 MB 가볍지만 대상 PC에 .NET 9 Desktop Runtime 필요(1회 설치) 내 PC들, 용량이 중요할 때
용량이 부담되면 1.6 MB 버전을 쓰고, 대상 PC에 .NET 9 데스크톱 런타임만 한 번 깔면 됩니다(무료, MS 공식). 처음 보는 PC라면 71 MB 자체 포함 버전이 가장 확실합니다. WPF 앱은 트리밍을 지원하지 않아 자체 포함을 더 줄이기는 어렵습니다.

옮겨서 실행하는 순서

  1. ScreenPenPortable.exe 한 개를 USB나 다른 PC 폴더(바탕화면 등)에 복사
  2. 더블클릭 → 바로 실행. 트레이(작업표시줄 우측)에 아이콘이 상주합니다.
  3. 설정 settings.json은 exe 옆에 자동 생성 → USB에 넣으면 설정도 함께 이동합니다.
⚠️ 처음 실행 시 SmartScreen 경고가 뜰 수 있습니다(서명 안 된 exe). "추가 정보 → 실행"을 누르면 됩니다(한 번만). 동작 환경은 Windows 10(1809+) / 11, 64비트입니다.

직접 빌드(개발자용)

dotnet publish src/ScreenPenPortable -c Release -r win-x64 --self-contained true `
  -p:PublishSingleFile=true -p:IncludeNativeLibrariesForSelfExtract=true `
  -p:EnableCompressionInSingleFile=true

런타임 의존(소형)으로 만들려면 --self-contained false 로 바꾸면 됩니다.

3. 화면 구성 — 플로팅 툴바

세로 2열 플로팅 툴바 (드래그로 이동 가능)

맨 위 점 6개(⠿) = 이동 핸들

이 부분을 잡고 끌면 툴바를 어디로든(다른 모니터 포함) 옮길 수 있습니다. 위치는 종료 시 저장됩니다.

  • ✏️ 펜 / 🖍️ 하이라이터 / ⌫ 지우개
  • ╱ 직선 / ↗ 화살표 / ▭ 사각형 / ◯ 타원 / ◑ 채움순환
  • T 텍스트 / ① 번호
  • 색상 스와치 (클릭으로 색 선택)
  • 굵기 슬라이더 (텍스트 도구일 땐 글자 크기)
  • ↩ 실행취소 / ↪ 다시 / 🗑 전체지우기
  • 📷 스크린샷 / ✋ 그리기·통과 전환
  • ⬜ 화이트보드 / 👻 고스트 / 🔍 돋보기 / 🌫 페이딩 / ◎ 하이라이트 / ⏱ 타이머
  • ✕ 종료
버튼이 파랗게 강조되면 현재 켜져 있는/선택된 상태입니다.

4. 그리기 도구

펜 · 하이라이터 · 지우개

  • 펜 Ctrl+Alt+1 — 자유 곡선. 색·굵기는 툴바에서.
  • 하이라이터 Ctrl+Alt+2 — 반투명 형광 강조.
  • 지우개 Ctrl+Alt+3 — 닿은 스트로크를 통째로 삭제.
  • 전체 지우기 Ctrl+Alt+E 또는 Delete — 모든 주석 제거(실행취소로 복구 가능).
도구마다 색·굵기를 따로 기억합니다(예: 펜은 빨강 3px, 하이라이터는 노랑 18px). 색은 스와치 클릭, 굵기는 슬라이더로 조절합니다.

5. 도형 · 텍스트 · 번호

도형 (직선/화살표/사각형/타원)

  • 툴바에서 도형을 고르고 드래그하면 그려집니다(드래그 중 미리보기 표시).
  • Shift 보정: 직선·화살표는 15° 각도 스냅, 사각형·타원은 정사각형·정원.
  • ◑ 채움 순환 버튼: 없음 → 컬러 채움 → 외곽선 → 흰색 채움 → 검정 채움 (사각형·타원에 적용).

텍스트 툴바 T

  • 클릭한 자리에 입력창이 열립니다. Enter 확정, Shift+Enter 줄바꿈, Esc 취소.
  • 텍스트 도구를 고르면 굵기 슬라이더가 글자 크기(8~200)로 바뀝니다.

번호 태그 툴바 ①

  • 클릭할 때마다 1, 2, 3… 번호가 매겨진 원형 태그를 찍습니다(절차 설명용).
  • 전체 지우기를 하면 번호가 다시 1부터 시작합니다.

6. 모드 & 효과

그리기 ↔ 통과(click-through) Ctrl+Alt+D

그리기 모드에서는 화면에 그림이 그려지고, 통과 모드에서는 입력이 아래 앱으로 통과되어 그려둔 주석을 유지한 채 데스크톱을 조작할 수 있습니다. (✋ 버튼으로도 전환)

고스트 모드 Ctrl+Alt+G

툴바를 숨기고 단축키만으로 사용합니다. 발표·녹화 시 화면을 깔끔하게. 돌아오기: Ctrl+Alt+G 다시 누르기

화이트보드 / 블랙보드 Ctrl+Alt+W

불투명 흰색/검정 배경을 깔아 그 위에 주석합니다. 누를 때마다 끄기 → 화이트 → 블랙 → 끄기 순환.

페이딩 잉크 Ctrl+Alt+F

그린 선이 잠시 후 자동으로 사라집니다(임시 강조용). 끄면 남아있던 선은 원래 진하기로 복원됩니다.

돋보기 / 줌 Ctrl+Alt+M

커서 주변 화면을 확대해 보여 줍니다. 같은 키나 🔍 버튼으로 끕니다(토글).

7. 타이머 Ctrl+Alt+C

화면 중앙 카운트다운 (메이플스토리체 Bold)

발표·시험·휴식 시간 표시용 큰 카운트다운 타이머입니다. 화면 정중앙에 표시됩니다.

  • 시간 설정: −1분 / +1분 / −10초 / +10초
  • 글자 크기: A− / A+ (변경해도 중앙 유지)
  • ▶ 시작 / ⏸ 일시정지 / ↺ 리셋 / ✕ 닫기
  • 0에 도달하면 빨간색 점멸 + 알림음
  • 설정한 시간·글자 크기는 저장됩니다.

패널 빈 곳을 드래그하면 위치를 옮길 수 있습니다(옮기면 자동 중앙정렬 해제).

8. 마우스 하이라이트(헤일로) Ctrl+Alt+H

마우스 커서 주위에 강조 원을 띄워 발표 중 포인터 위치를 부각합니다. 같은 키나 ◎ 버튼으로 켜고 끕니다.

색상 · 크기 바꾸기

  • 툴바의 ◎ 버튼을 우클릭 하면 설정창이 열립니다.
  • 색상 10종 스와치 + 크기 슬라이더(8~100px)
  • 바꾸면 즉시 적용되고 설정에 저장됩니다.
◎ 우클릭 → 색상·크기 설정

10. 팁 & 주의

  • 그리기 모드에선 다른 앱 클릭이 안 됩니다. 오버레이가 화면 입력을 잡기 때문 — Ctrl+Alt+D로 통과 모드로 바꾸면 아래 앱을 조작할 수 있고 그려둔 주석은 유지됩니다.
  • 종료는 Ctrl+Alt+Q · 트레이 아이콘 우클릭 · 툴바 ✕. (Esc는 실수 종료 방지를 위해 더 이상 앱을 닫지 않습니다.)
  • 멀티모니터: 모든 화면에 그릴 수 있고, 모니터를 추가/제거하면 자동으로 재배치됩니다.
  • 설정(색·굵기·핫키 결과·툴바 위치·타이머·하이라이트 등)은 exe 옆 settings.json에 저장되어 USB로 함께 이동합니다.
  • 일부 단축키가 다른 앱과 겹쳐 등록에 실패하면 트레이 아이콘 툴팁에 알려줍니다.
